# 鲜小智生鲜买菜系统

> 一个基于 SpringBoot + 微信小程序原生 + Vue 管理后台 的生鲜购物超市。具体项目设计请看：需求分析.md

### ✨ 技术栈
后端：


- **核心框架**: Spring Boot 2.7.x / 3.x
- **项目管理**: Maven / Gradle
- **数据库**: MySQL 8.0 + MyBatis-Plus / Spring Data JPA
- **缓存**: Redis (用于 token 存储/分布式锁)
- **工具**: Lombok, Hutool, Swagger/Knife4j (API 文档)

### 🚀 快速启动

### 前置要求
- JDK 17+
- Maven 3.6+
- MySQL 8.0+
- Redis (可选)
